# Media Application

Spring Boot application demonstrating Spring Batch and Spring Shell integration.

## Prerequisites

- Java 25
- Maven 3.x

## Building the Application

To build the application, run:

```bash
mvnw clean package
```

Or on Windows:

```bash
mvnw.cmd clean package
```

## Running the Application

### Run with Maven

```bash
mvnw.cmd spring-boot:run -Dspring-boot.run.arguments="start"
```

### Data structure
The application models and manages structure of media files located in static, well structurized folders.
